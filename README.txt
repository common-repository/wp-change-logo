== Description ==

This plugin helps you to customize wordpress defualt login page. You can change logo and body background color, text color etc.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-change-logo.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How can I change logo on login page? =

Just go to admin dashboard->logo setting page and past the uploaded image url in textbox and save settings.

= Can I change page background color? =

Just go to admin dashboard->logo setting page and select the color from color picker and save settings.
